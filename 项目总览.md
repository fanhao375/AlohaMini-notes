## 这台机器人的核心参数（速查；权威以 `AlohaMini/AlohaMini2/docs/BOM.md` 为准）

| 参数 | AlohaMini2 |
|------|-----------|
| 形态 | 双臂 + 电动升降 + 三全向轮移动底盘 |
| 自由度 | **18**（双臂 6+1 ×2 = 14 + 3 轮 + 1 升降） |
| 执行器 | **飞特 Feetech STS 总线舵机**（非力矩电机）：STS-3215(12V) / STS-3095(12V 95kg·cm)；Pro 加 STS-3250 |
| 控制 | 串行总线舵机协议 → Waveshare 总线舵机控制器 → 树莓派 5 |
| 架构 | LeKiwi Client/Host：Host(树莓派在车上) ↔ Client(PC) 走 WiFi |
| 臂展 / 负载 | 52cm / 1kg（单臂） |
| 自建成本 | ~$1000 |
| 算法 | 原生 LeRobot；ACT / Diffusion Policy / SmolVLA / Pi-0.5(openpi) |

**路线定位**：便宜总线舵机 + 位置控制 + 3D 打印（SO-ARM/LeRobot 一系），靠低成本与数量取胜。
对比 reBot-DevArm = 达妙/RobStride 力矩电机 + CAN + 力控（高端力控路线）。
